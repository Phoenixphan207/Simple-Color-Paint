# Simple-Color-Paint

A very simple web-based drawing app with 7 colors and a clean color palette.

## Features
- Draw with mouse (click + drag)
- 7 colors: Red, Blue, Green, Yellow, Purple, Orange, Black
- Horizontal color palette at the top
- Fixed line thickness (6px), smooth lines
